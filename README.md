📌 Project Title

Student Performance Analysis Using Statistical Methods and AI Techniques

📖 Overview

This project focuses on analyzing student academic performance using statistical methods and artificial intelligence techniques. The study is based on a dataset of 127 students, including their academic scores from previous and current semesters along with study-related factors.

The objective is to analyze performance trends, identify improvement or decline, and apply machine learning models to predict student outcomes.

🎯 Objectives
To analyze student performance using statistical methods
To compare performance across semesters
To measure improvement or decline
To identify factors affecting performance
To apply machine learning techniques for prediction
📊 Dataset Description

The dataset consists of 127 student records collected using Google Forms.

🔹 Attributes:
Last Semester MST Percentage
Current Semester MST Percentage
Study Hours
Attendance
Revision Habits
Performance Change (calculated)
📈 Methodology
🔹 1. Data Collection

Data was collected through an online survey form and organized into a structured dataset.

🔹 2. Data Preprocessing
Converted marks into percentage format
Encoded categorical variables
Calculated performance change:
Change=Current−Last Semester
🔹 3. Statistical Analysis
Mean
Median
Variance
Standard Deviation
🔹 4. Data Visualization
Histogram
Bar Graph
Pie Chart
Scatter Plot
🔹 5. Correlation Analysis

Used to identify relationships between:

Study hours and performance
Attendance and marks
🤖 Machine Learning Models
🔹 1. Decision Tree

A Decision Tree is a supervised learning algorithm used for classification and regression. It splits the dataset into branches based on feature values and provides easy-to-understand rules.

👉 Used in this project to:

Classify student performance (Improved / Declined)
Understand decision patterns
🔹 2. Random Forest

Random Forest is an ensemble learning technique that builds multiple decision trees and combines their outputs to improve accuracy and reduce overfitting.

👉 Used in this project to:

Improve prediction accuracy
Handle dataset efficiently
Provide reliable results
📊 Model Comparison
Decision Tree provides simple and interpretable results
Random Forest gives higher accuracy and stability

The Random Forest model performed better, showing improved prediction accuracy for student performance.

🧠 Case Study Summary

The analysis showed that students who increased study hours and maintained higher attendance demonstrated improvement. Students with irregular study patterns showed decline.

📊 Results
Majority of students showed improvement
Positive correlation between study hours and marks
Attendance significantly affects performance
Random Forest outperformed Decision Tree
